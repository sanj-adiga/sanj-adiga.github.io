Autonomous Vehicle Lane Detection

A deep learning model using transfer learning to segment and classify lane markings in real-time driving scenarios.

Built a hybrid CV + deep learning system to detect lane boundaries in autonomous driving scenarios. I used Canny/Sobel filters and Hough Transforms for classical edge work, then trained a VGG16-UNet on the TuSimple dataset, reaching 97.72% validation accuracy. My pipeline emphasized clarity: grayscale, ROI masking, brightness normalization — small things that made the model more confident. What stuck with me: even the most “accurate” models still need to think about generalization. Complex visibility conditions are where machine perception really starts to matter.

Python TensorFlow Keras OpenCV U-Net VGG-16 Computer Vision Image Segmentation Edge Detection
